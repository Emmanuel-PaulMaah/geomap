# geo-command-center

a geographic command center built with react, leaflet, & vite.

## what it does

displays interactive maps & geographic data visualization using leaflet & react-leaflet. built with modern tooling for fast development & deployment.

## setup

```bash
npm install
```

## development

```bash
npm run dev
```

runs on http://localhost:5173

## build

```bash
npm run build
```

outputs to `dist/` directory

## lint

```bash
npm run lint
```

checks code with eslint

## preview

```bash
npm run preview
```

preview production build locally

## tech stack

- **react** 19 - ui framework
- **leaflet** 1.9 - mapping library
- **react-leaflet** 5 - react bindings for leaflet
- **lucide-react** 1.7 - icon library
- **vite** 8 - build tool
- **eslint** 9 - code linting

## structure

```
src/           - source code
dist/          - production build (generated)
DOCS/          - documentation
index.html     - html entry point
vite.config.js - vite configuration
```
